# Go-Weather

Go-weather is an educational project and is supposed t be used ass the same, This application is a weather application which fetches its data from an API, it requires an input (city name) from the user and it will show the following data
1. Temprature
2. Heighest temprature
3. Lowest temprature
4. Feels like
5. Wind temprature
6. Wind speed
And many more.

The data is fetched using an API from RapidAPI
